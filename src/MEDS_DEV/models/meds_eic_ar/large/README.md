# MEDS-EIC-AR (large)

The **large** capacity variant of [MEDS-EIC-AR](https://github.com/mmcdermott/MEDS_EIC_AR):
a 24-layer transformer (16 attention heads x 64-dim, hidden size 1024),
selected via the upstream `lightning_module=large` config preset.

See the [parent README](../README.md) for the shared pipeline description, the demo/full
behavior, and the current state of zero-shot prediction support.
