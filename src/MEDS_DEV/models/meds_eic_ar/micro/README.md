# MEDS-EIC-AR (micro)

The **micro** capacity variant of [MEDS-EIC-AR](https://github.com/mmcdermott/MEDS_EIC_AR):
a 4-layer transformer (4 attention heads x 64-dim, hidden size 256),
selected via the upstream `lightning_module=micro` config preset.

See the [parent README](../README.md) for the shared pipeline description, the demo/full
behavior, and the current state of zero-shot prediction support.
