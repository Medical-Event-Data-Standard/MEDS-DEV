# MEDS-EIC-AR (medium)

The **medium** capacity variant of [MEDS-EIC-AR](https://github.com/mmcdermott/MEDS_EIC_AR):
a 12-layer transformer (12 attention heads x 64-dim, hidden size 768),
selected via the upstream `lightning_module=medium` config preset.

See the [parent README](../README.md) for the shared pipeline description, the demo/full
behavior, and the current state of zero-shot prediction support.
