# MEDS-EIC-AR (small)

The **small** capacity variant of [MEDS-EIC-AR](https://github.com/mmcdermott/MEDS_EIC_AR):
a 8-layer transformer (8 attention heads x 64-dim, hidden size 512),
selected via the upstream `lightning_module=small` config preset.

See the [parent README](../README.md) for the shared pipeline description, the demo/full
behavior, and the current state of zero-shot prediction support.
